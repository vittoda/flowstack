## About the system : Google Calendar Event Creation
- The system supports creating calendar events via a single operation:
  - createEvent
- This operation is used to schedule events with relevant details such as:
  - Title (event summary)
  - Description
  - Start date and time
  - End date and time
  - Time zone (if applicable)
  - Attendees (email addresses)

## Operation Specification

### createEvent
Use this operation to create a calendar event.

#### Required Fields:
- summary: Title of the event
- start: Start date and time (ISO 8601 format preferred)
- end: End date and time (ISO 8601 format preferred)

#### Optional Fields:
- description: Additional details about the event
- location: Event location (e.g., meeting room, Zoom link)
- attendees: List of participant email addresses
- timeZone: Time zone of the event (e.g., "Asia/Kolkata")
- reminders: Custom reminders (e.g., email or popup before event)
