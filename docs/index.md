# FlowStack - Agent Orchestration Framework

FlowStack is a lightweight AI agent orchestration framework designed for flexibility and ease of integration. It can be embedded into existing applications or used as a standalone platform. Built with observability in mind, FlowStack includes integrated logging and debugging capabilities from the ground up, making it easier to monitor, trace, test, and troubleshoot agent workflows.
