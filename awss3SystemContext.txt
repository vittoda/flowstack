## About the system : AWS S3 File Upload
- The system supports uploading files via a single operation:
  - putFile
- This operation is used to upload files with relevant details such as:
  - Bucket name
  - Object key (file name/path in S3)
  - File content (or reference/path)
  - Content type (if applicable)

## Operation Specification

### putFile
Use this operation to upload a file to an AWS S3 bucket.

#### Required Fields:
- bucketName: Name of the target S3 bucket
- objectKey: Name (and optional path) of the object in the bucket
- fileContent: Content of the file or a valid reference/path to the file

#### Optional Fields:
- contentType: MIME type of the file (e.g., "application/pdf", "image/png")

## Execution Assumptions
- All files are uploaded to the specified S3 bucket.
- Object keys act as unique identifiers within a bucket.
- File size must be less than 5 MB.
- Ensure the bucket exists and is accessible.
- Ensure file content is valid and not empty.
